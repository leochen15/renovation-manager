# Configuration Reference

## Environment Variables
- `EXPO_PUBLIC_SUPABASE_URL`
- `EXPO_PUBLIC_SUPABASE_ANON_KEY`
- `EXPO_PUBLIC_USE_MOCKS` (set to `true` to run offline UI)

## Optional Defaults (editable in code)
- Gantt day width: `src/screens/ScheduleScreen.tsx` (`dayWidth`)
- Theme colors: `src/styles/theme.ts`

## App Identity
- `app.json` (name, slug, icon, splash)
- `package.json` (npm package name)
